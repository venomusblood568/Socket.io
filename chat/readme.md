# Chat App

A simple chat application built with Node.js and Socket.IO, allowing users to join different chat rooms and communicate in real-time.

## Features
- [x] basic setup done
- [x] realtime update
- [x] bot for notification (in and out of user) 
- [x] simple chat 
- [x] username
- [x] room
- [x] custom room
- [x] better ui
- [] room limit
